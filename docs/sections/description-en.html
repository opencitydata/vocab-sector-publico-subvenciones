Vocabulary for the representation of grants awarded by a municipality: Description back to ToC

The following figure shows the class diagram of the ontology, where it can be seen that the core element is the class [essubv:Subvencion](http://vocab.ciudadesabiertas.es/def/sector-publico/subvencion#Subvencion), defined in this vocabulary:
The diagram revolves around the core class [essubv:Subvencion](#Subvencion), which is the main class in this vocabulary. This class will be related to the class [org:Organization](#Organization) using: - [essubv:beneficiario](#beneficiario), which would be the awarded of a [essubv:Subvencion](#Subvencion), either a natural or legal person. - [essubv:entidadFinanciadora](#entidadFinanciadora), which would correspond to the [org:Organization](#Organization) that finances the [essubv:Subvencion](#Subvencion). - [essubv:area](#area), which would correspond to the [essubv:area](#area) that awards the [essubv:Subvencion](#Subvencion). - [essubv:servicio](#servicio), which would correspond to the [essubv:servicio](#servicio) of the city council that offers the [essubv:Subvencion](#Subvencion). - [essubv:gestionadoPor](#gestionadoPor), which would correspond to the [org:Organization](#Organization) that manages the [essubv:Subvencion](#Subvencion), which can be a government area or a [esadm:Distrito](#Distrito). Finally, the class [essubv:Subvencion](#Subvencion) can arrange a [esconv:Convenio](#Convenio) or be implemented by it. In addition, in this vocabulary several SKOS concept schemes have been used to represent: - Types of instrument - Types of procedure - Spending program classification - Economic classification of expenditure - Grant topics In Figure 1 several relationships with the SKOS concept class can be observed, however, the specific values have not been specified as they are different depending on the taxonomy they represent. For example, the property [essubv:tipoProcedimiento](#tipoProcedimiento) represents whether the grant is of type "direct-grant" or "concurrency".

Use cases volver a índice

In order to facilitate the understanding of the use of some classes and properties of this vocabulary, some examples will be provided below. First, to represent a grant, an instance of the class[essubv:Subvencion](#Subvencion) is created with the properties [dct:identifier](http://purl.org/dc/terms/identifier) (grant identifier), [dct:title](http://purl.org/dc/terms/title) (grant title), [essubv:tieneTematica](#tieneTematica) (theme of the grant), [essubv:tipoInstrumento](#tipoInstrumento) (grant tool), [essubv:beneficiario](#beneficiario) (beneficiary of the grant), [essubv:importeConcedido](#importeConcedido) (final amount of the grant) y [essubv:fechaAdjudicacion](#fechaAdjudicacion) (grant award date).
